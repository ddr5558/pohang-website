# 프로젝트 맥락: 포항 기업혁신파크 소개 웹사이트

> 이 파일은 앞으로 만들 모든 문서(프롬프트, 사이트 구조, 랜딩 페이지, 검색엔진 안내 파일,
> 콘텐츠 자동화 워크플로우)의 톤과 주제를 맞추는 기준입니다.

## 1. 사이트 목적
- 포항 기업혁신파크를 소개하고, 입주·협력·방문 문의를 자연스럽게 유도하는 공식 소개 사이트
- 검색엔진에서 "포항 기업혁신파크", "포항 창업 입주", "포항 기업 지원" 등으로 잘 발견되도록 구성

## 2. 대상 독자
- **1순위**: 입주를 고려하는 창업자 · 중소/스타트업 기업 담당자
- **2순위**: 지역 혁신 · 산업 클러스터에 관심 있는 기관, 연구자, 일반 시민
- **3순위**: 협력 파트너(투자사, 지원기관, 대학 등)

## 3. 톤앤매너
- 신뢰감 있고 명료한 공공/기관 톤 (과장된 홍보 문구 지양)
- 핵심 정보를 빠르게 전달하는 구조 (입주 혜택, 시설, 위치, 문의 경로)
- 전문적이되 딱딱하지 않게, 따뜻한 지역 브랜드 인상

## 4. 주요 콘텐츠 주제
- 혁신파크 소개 (비전, 규모, 위치, 주요 시설)
- 입주 안내 (대상, 혜택, 절차, 자격)
- 지원 프로그램 (창업 지원, 네트워킹, 인프라)
- 입주 기업 / 성공 사례
- **공지·소식 게시판** (아래 별도 명시)

## 5. 관리자 게시판 요구사항
- 관리자가 게시글을 **업로드 / 수정 / 삭제**할 수 있는 공지·소식 게시판
- 일반 방문자는 게시글을 **열람만** 가능
- 게시글 항목: 제목, 본문, 작성일, 카테고리(공지/소식/모집 등)
- (향후) 관리자 로그인 인증, 첨부파일, 검색 기능 확장 고려
- 관리자가 사이트 사진/이미지(배너, 시설 사진, 사례 이미지 등)를 직접 업로드·교체·삭제 가능
- 이미지 교체 시 SEO용 alt 텍스트도 함께 입력/수정

## 6. 필수 SEO 요소 (이후 단계에서 반영)
- 메타 태그(title, description)
- 스키마 마크업(JSON-LD) — Organization / LocalBusiness 성격
- sitemap.xml, robots.txt
- 다국어(국문/영문) 확장 가능성 고려

## 7. 콘텐츠 자동화 방향
- 게시판 원고를 블로그 글 · SNS 게시물 · 보도자료 초안으로 재활용
- 발행 전 사람이 검토하는 수동 승인 단계 유지
