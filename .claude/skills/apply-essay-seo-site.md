# 회사 적용: 포항 기업혁신파크 SEO 사이트

"철학 에세이 SEO 사이트 초안" 실습 결과물(포항 기업혁신파크로 진행)을 실제 데이터에 적용하는 가이드입니다.
한국어 존댓말. 각 [STOP]에서 멈추세요. 학습자가 중단을 원하면 진행분까지만 기록하고 마무리.

## 진행 체크리스트

- [ ] 1단계: 실제 데이터 준비 완료
- [ ] 2단계: 데이터 교체 완료
- [ ] 3단계: 실행 검증 통과
- [ ] 4단계: 정기 실행 설정 (선택)

> 각 단계 완료 시 체크리스트를 업데이트하세요 (Edit 도구로 `[ ]`를 `[x]`로 변경).

## 1단계: 실제 데이터 준비

**[필수]** 이 가이드는 실제 데이터로 진행합니다. 샘플/더미 데이터는 사용하지 마세요.

| 대상 | 실습에서 사용한 값 | 회사 적용 방법 |
|------|-------------------|--------------|
| 사이트 주제 | 포항 기업혁신파크 소개 | 실제 혁신파크 명칭·슬로건·소개 문구와 키워드로 교체합니다. |
| GNB 메뉴 | 넥스폴리스 구조 차용(소개/개발현황/분양·입주안내/홍보센터/회사소개) | 실제 운영 메뉴와 서브페이지에 맞게 항목·링크를 정리합니다. |
| 히어로 배경 | unsplash 임시 도시 사진 | 실제 혁신파크 항공뷰/조감도 사진으로 교체하고 alt 텍스트를 입력합니다. |
| 성과 지표 | 120+/500억+/1,200+/30+ (예시) | 실제 입주기업 수·투자·고용·프로그램 수치로 교체합니다. |
| 도메인 및 사이트맵 | https://example.com 기반 예시 | 실제 도메인으로 교체하고 sitemap.xml/robots.txt URL을 갱신합니다. |
| 콘텐츠 재활용 흐름 | 블로그·인스타·링크드인·보도자료·내부 링크 | 운영 채널에 맞춰 뉴스레터·FAQ 등으로 확장합니다. |

"위 표에서 교체할 실제 데이터/사진/파일이 있으신가요? 파일 경로를 알려주세요."

**[STOP]** 학습자가 실제 파일 경로를 알려줄 때까지 기다리세요.
- 학습자가 "아직 없어요"라고 하면: 어떤 형식이 필요한지 안내하고, 준비한 뒤 다시 오라고 안내하세요.
- **절대 샘플 데이터를 생성하지 마세요.**

## 2단계: 데이터 교체

샘플 도메인·예시 수치·임시 사진을 실제 도메인, 실재 성과 수치, 실제 사진으로 교체합니다.
입주기업명과 성과 수치는 원자료와 대조해 정확히 반영합니다.

**지원 형식**: md, html, css, xml, txt, csv, (이미지) jpg/png/webp

학습자가 알려준 데이터에 맞게 함께 진행하세요:
example.com → 실제 URL, 예시 수치 → 실제 수치, 임시 배경 이미지 → 실제 사진(alt 포함), 메뉴 링크 검수.

**[STOP]** 데이터 교체가 완료될 때까지 함께 진행하세요.

## 3단계: 결과 검증

교체한 데이터로 확인합니다:
배포(또는 로컬 열람) 후 title, meta description, JSON-LD, sitemap.xml, robots.txt, 이미지 alt가 올바른지 점검합니다.

**검증 기준:**
- 에러 없이 페이지가 정상 렌더링됨
- 출력에 실제 데이터가 반영됨 (샘플 흔적 없음)
- 결과 파일이 정상 생성/갱신됨

에러가 발생하면 함께 디버깅하세요.

**[STOP]** 위 3가지 검증 기준을 모두 통과할 때까지 함께 진행하세요.

## 4단계: 정기 실행 (선택)

"콘텐츠 생성을 자동으로 돌아가게 설정할 수 있어요. 해볼까요?"

**방법**: cron 또는 GitHub Actions
**권장 주기**: 주 1회 또는 콘텐츠 발행 일정에 맞춰
정기적으로 게시판 원고를 읽어 블로그/소셜 초안을 생성한 뒤, 수동 승인 후 배포하도록 구성합니다.

```
0 9 * * 1 cd /path/to/project && python generate_content.py >> logs/automation.log 2>&1
```

**[STOP]** 설정 완료 또는 건너뛰기를 기다리세요.
