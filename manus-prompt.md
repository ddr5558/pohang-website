## 제작 요청문 (복사해서 사용)

포항 기업혁신파크를 소개하는 웹사이트를 만들어주세요.

**목표 독자**
- 1순위: 입주를 고려하는 창업자 · 중소/스타트업 기업 담당자
- 2순위: 지역 혁신 · 산업 클러스터에 관심 있는 기관, 연구자, 시민
- 3순위: 협력 파트너(투자사, 지원기관, 대학)

**페이지 구성**
- 홈(핵심 가치 + CTA)
- 혁신파크 소개(비전, 규모, 위치, 시설)
- 입주 안내(대상, 혜택, 절차, 자격)
- 지원 프로그램(창업 지원, 네트워킹, 인프라)
- 입주 사례 / 성공 스토리
- 공지·소식 게시판 (관리자 전용 업로드/수정/삭제, 방문자는 열람만)
- 문의(연락처, 위치 지도, 문의 폼)

**콘텐츠 블록**
- 입주 혜택 비교 테이블
- 국문/영문 다국어 섹션(확장 가능 구조)
- 입주 절차 단계 안내
- 자주 묻는 질문(FAQ)

**필수 SEO 요소**
- 페이지별 메타 태그(title, description)
- JSON-LD 스키마 마크업(Organization / LocalBusiness 성격)
- sitemap.xml(주요 페이지 포함)
- robots.txt(Sitemap 경로 포함)

**관리자 게시판 요구사항**
- 관리자가 게시글을 업로드 / 수정 / 삭제할 수 있는 권한
- 일반 방문자는 열람만 가능
- 게시글 항목: 제목, 본문, 작성일, 카테고리(공지/소식/모집)
- 향후 관리자 로그인 인증 · 첨부파일 · 검색 기능 확장 고려

**관리자 이미지 관리 요구사항**
- 관리자가 사이트에 노출되는 사진/이미지를 직접 업로드 · 교체 · 삭제할 수 있는 권한
- 대상: 홈 배너/히어로 이미지, 소개·시설 사진, 입주 사례 이미지, 게시글 첨부 이미지
- 이미지 교체 시 SEO를 위한 alt 텍스트(대체 텍스트)도 함께 입력/수정 가능하게
- 일반 방문자는 이미지 열람만 가능

**콘텐츠 자동화 요구사항**
- 게시판에 올린 공지·소식 원고를 블로그 글, SNS(인스타그램/링크드인) 게시물 초안,
  보도자료 초안으로 재활용하는 워크플로우를 제안해주세요.
- 발행 전 사람이 검토하는 수동 승인 단계를 포함해주세요.

**톤앤매너**
- 신뢰감 있고 명료한 공공/기관 톤, 과장된 홍보 문구 지양
- 핵심 정보를 빠르게 전달하는 깔끔한 구조
